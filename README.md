# flask-helloworld
A simple Flask app

## Dependencies
- Python
- Flask

## How to run
* Clone the repo
* $ cd flask-helloworld
* $ export FLASK_APP=app.py
* $ flask run

## Flask recipes
* **app.py :**  simple hello world flask app
* **return_json.py :** return JSON output from a flask app
* **handle_postdata.py :** handle post data in json format
